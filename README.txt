GeoConflicClient - Luxon

 Proyecto: GeoConflit
 Descripcion: Cliente para android que recibe de la forma más rapida posible la ubicacion obtenida del GPS.
 Mostrando el pantalla de forma gráfica una ruta de la traza del camino que el usuario realiza desde la primera
  posición registrada por el GPS.
 
 Version: 0.0 23/08/2013
 
 Requisitos:
 	- Java SDK 6
 	- Android SDK, target 2.3.3, api level 10
